如果你是新用户，建议从 [基本使用示例](basic-usage/) 开始。

## 💡 实用技巧

1. **从简单开始**: 先配置基本的分类规则，逐步添加复杂条件
2. **测试验证**: 使用小量数据测试规则是否符合预期
3. **备份配置**: 保存好工作的配置文件，方便复用和修改
4. **持续优化**: 根据实际使用情况不断完善规则
